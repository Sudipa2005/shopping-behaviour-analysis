# Customer Shopping Behaviour Analysis

## 📌 Project Overview

This project analyzes customer shopping behaviour data to identify purchasing trends, customer demographics, and product category performance.

## 🛠 Tools & Technologies

* Python (Pandas, Matplotlib, Seaborn)
* SQL (SQLite for querying insights)
* Power BI (Interactive dashboard visualization)

## 📊 Key Analysis Performed

* Gender-wise spending analysis
* Category-wise sales distribution
* Seasonal purchasing trends
* Age vs Purchase behaviour analysis
* Customer segmentation insights

## 📈 Dashboard Insights

* Clothing category generated highest revenue
* Male customers contributed more to total spending
* Winter and Fall seasons showed peak sales trends
* Customers aged 20–40 showed higher purchasing frequency
